<h1 align="center" id="title">Kotlin E-Commerce</h1>

<h2>Features</h2>

Here're some of the project's best features:

*   Retrofit
*   Rest-Api
*   Glide
*   Fragment
*   Navigation Component
*   MVVM
*   Firebase Cloud Messaging
*   LiveData
*   Singleton
*   Object Oriented Programming
*   Custom UI
*   DummyJson
